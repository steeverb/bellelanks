Four directions, one decision.

Same content, four different ways of presenting it. Open whichever you'd like to react to — or all of them, in any order. Notes and tradeoffs are in the README.

  • 04 Editorial · hybrid Recommended Editorial — with case study sub-pages Magazine sensibility. Each major project gets its own page (Samsung Explore, Citi Mortgage, New York Life, Magenta, Disrupt Aging) with a real role/client/output meta strip and room to breathe. →
  • 01 Refresh + Modernize Closest to the current site Faithful upgrade of the existing card grid. Modern type, real accent color, light/dark mode. Lowest cognitive load for returning visitors. →
  • 02 Editorial · serif-led A magazine, on one page All on one URL: cover, drop-cap featured case, byline grid, typographic masthead. Same DNA as Direction 4 but no sub-pages. →
  • 03 Minimal · typographic Type does the work No hero image. Mono-driven, list-based, voice-forward. Heavy whitespace. Of-the-moment if the audience skews tech/design-product. →
